### My projects

**[go-md2pdf](https://github.com/alnah/go-md2pdf)**: Go library and CLI for converting Markdown to PDF. Cover pages, automatic table of contents, watermarks, signatures, CSS themes, and custom template support. Parallel batch processing via headless Chrome. No LaTeX. No complexity.

**[go-transcript](https://github.com/alnah/go-transcript)**: Go CLI to transcribe meetings, lectures, and brainstorms into structured Markdown. Uses OpenAI for transcription and DeepSeek/OpenAI for AI-powered formatting.

**[things-agent](https://github.com/alnah/things-agent)**: Go CLI that enables AI coding agents such as Codex and Claude Code to control Things 3 through a local wrapper around AppleScript and the Things 3 URL scheme, without direct SQLite access.
